# Presentation Navi

プレゼンテーション作成支援ツール

## オフラインでの使用方法
このプロジェクトは、インターネット接続がない環境（オフライン）でも動作するように設定されています。

### 起動方法
フォルダ内の `スタート.html` をダブルクリックして開いてください。

### 必要な準備（初回のみ）
以下のコマンドを実行して、必要なファイルをインストールします（Node.jsが必要です）。

```bash
npm install
```

### CSSのビルド（デザイン調整時）
デザイン（`style.css`やHTML）を変更した場合は、以下のコマンドを実行してCSSを再構築してください。

```bash
npm run build
```

### オンラインモード（CDN）への切り替え
開発の都合などでCDN（インターネット経由）を使用したい場合は、各HTMLファイル内の以下の部分を変更してください。

```html
<!-- Local CSS (Offline) -->
<!-- <link rel="stylesheet" href="output.css"> --> <!-- コメントアウト -->
<!-- CDN (Online) -->
<script src="https://cdn.tailwindcss.com/3.4.16"></script> <!-- コメント解除 -->
```

## 注意事項
-   AI添削機能はインターネット接続が必要です。オフライン時はエラーメッセージが表示されます。
